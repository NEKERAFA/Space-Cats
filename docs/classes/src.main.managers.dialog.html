Methods

src.main.managers.dialog:init (self, delay, auto, codes)
Create a new dialog system

Parameters:

  • self dialog New dialog object
  • delay number Delay between words
  • auto boolean Set dialogs that change automatic
  • codes table Table with scancodes to control dialogs, where 'next' key has scancode to pass dialog and 'auto' key has scancode to put dialogs automatic
src.main.managers.dialog:add_line (self, title, message)
Add new character line to script

Parameters:

  • self dialog Dialog system
  • title string Title of new message. Put the name of charater
  • message string The text that character said
src.main.managers.dialog:add_text (self, message)
Add more text to script. Use this if you can add more lines to a character.

Parameters:

  • self dialog Dialog system
  • message string The text that character said
src.main.managers.dialog:keypressed (self, key, scancode, isrepeat)
Update menu variable

Parameters:

  • self dialog Dialog system
  • key KeyConstant Character of the pressed key
  • scancode Scancode The scancode representing the pressed key
  • isrepeat bolean Whether this key press event is a repeat. The delay between key depends on the user's system settings
