Module Ship

Ship prototype object.

This module construct a ship object that have this variables ship.x : x screen position ship.y : y screen position ship.life : current life ship.ship_type : Type of current ship ship.bullets : List of shot bullets ship.bullet_threshold : Time between bullets ship.flame : Animation of flame ship.explosion : Animation of explosion
