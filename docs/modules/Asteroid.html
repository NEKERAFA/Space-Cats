Module Asteroid

Asteroid prototype object.

This module construct a asteroid object that have this variables asteroid.x : x screen position asteroid.y : y screen position asteroid.vx : x velocity of asteroid asteroid.vy : y velocity of asteroid asteroid.strength : Level of damage resistance asteroid.asteroid_type : Type of current asteroid (randomized) asteroid.explosion : Animation of explosion asteroid.damage : Return if asteroid is damaged asteroid.collider : Save the collinder of asteroid
